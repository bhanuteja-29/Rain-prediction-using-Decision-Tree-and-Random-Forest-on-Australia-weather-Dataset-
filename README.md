# Rain-prediction-using-Decision-Tree-and-Random-Forest-on-Australia-weather-Dataset-

This project involves developing a machine learning model to predict rainfall in Australia using the Random Forest algorithm. The model analyses historical weather data to identify patterns and relationships between meteorological features and rainfall occurrence. By leveraging the ensemble learning approach of Random Forest, the model achieves improved accuracy and robustness compared to single models, helping support weather forecasting and decision-making applications.
